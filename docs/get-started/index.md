# Get started with Renegade Kernel

Welcome to Renegade Kernel! This guide will help you install, configure, and start
using the Renegade Kernel to enhance your workflow right from your terminal.

## Quickstart: Install, authenticate, configure, and use Renegade Kernel

Renegade Kernel brings the power of advanced language models directly to your command
line interface. As an AI-powered assistant, Renegade Kernel can help you with a
variety of tasks, from understanding and generating code to reviewing and
editing documents.

## Install

The standard method to install and run Renegade Kernel uses `npm`:

```bash
npm install -g @zombiegirlcz/RENEGADEkarnelcli
```

Once Renegade Kernel is installed, run Renegade Kernel from your command line:

```bash
renegade
```

For more installation options, see [Renegade Kernel Installation](./installation.md).

## Authenticate

To begin using Renegade Kernel, you must authenticate with a Google service. In most
cases, you can log in with your existing Google account:

1. Run Renegade Kernel after installation:

   ```bash
   renegade
   ```

2. When asked "How would you like to authenticate for this project?" select **1.
   Login with Google**.

3. Select your Google account.

4. Click on **Sign in**.

Certain account types may require you to configure a Google Cloud project. For
more information, including other authentication methods, see
[Renegade Kernel Authentication Setup](./authentication.md).

## Configure

Renegade Kernel offers several ways to configure its behavior, including environment

Once installed and authenticated, you can start using Renegade Kernel by issuing
